# Personal Portfolio — Ridam Rabi Das

A clean, minimal portfolio built with React and Tailwind CSS.  
Live at: **[\[ridamrabidas.vercel.app\](https://ridamrabidas.vercel.app)](https://portfolio-nu-gilt-76.vercel.app/)**

---

## Stack

- **React** — component structure and routing
- **Tailwind CSS** — styling
- **Vite** — build tooling
- **Vercel** — deployment

---

---

## Running locally

```bash
git clone https://github.com/ladridam/portfolio
cd portfolio
npm install
npm run dev
```
